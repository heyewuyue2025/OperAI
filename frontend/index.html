OperAI X Harness

OperAI 是面向运营团队的 AI 工作平台。内容运营、用户运营、活动运营、渠道运营等职能都能从自己的入口进入；系统在背后用 Harness 编排 Skill，完成理解、生成、复核和交付。

职能入口

先找到你的工作，
再交给系统编排。

运营用户看到的是自己熟悉的岗位场景；Harness 负责在背后选择 Skill、传递上下文、记录过程和检查质量。

内容内容运营

把素材、卖点和用户语境转成微博、公众号、小红书、抖音内容。

用户用户运营

把反馈、分层和生命周期问题转成触达、召回和留存动作。

活动活动运营

把目标、预算、资源和节奏组织成可执行活动方案。

渠道渠道运营

把内容安排到合适平台、时间窗口、标签和发布节奏。

增长增长转化

围绕流量效率、转化路径、促销和 CTA 给出优化动作。

产品产品运营

把功能数据、体验问题和用户反馈转成迭代优先级与运营动作。

社群社群运营

把社群聊天、评论区和 KOL 线索转成互动话术与社群动作。

市场市场策略

把品牌、竞品、趋势和目标客群整理成定位与传播判断。

质量检验

好答案不是写出来，
是被检查出来。

Harness 会记录 Skill 选择、上下文传递、质量检查、证据缺口和导出状态，让运营团队知道结果从哪里来、为什么可信。

收到任务 / 系统先读懂目标、平台、受众和限制条件
匹配能力 / 自动挑选合适的运营智能体和 Skill 组合
生成方案 / 产出内容、排期、触达动作或转化建议
复核交付 / 检查依据、风险和完整度，整理成可提交版本

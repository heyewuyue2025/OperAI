Archive OS 把运营工作拆成五类可追溯对象。它保留 AI 的生成能力，但让结果有证据、有来源、有运行轨迹。

01Task File

输入素材、标题、平台、Pack 与 DAG，形成可运行案卷。

02Agent Index

十个运营 Agent 作为可检索研究工具，而不是普通菜单。

03Run Dossier

每次执行都记录步骤、状态、耗时、输出和降级标记。

04Evidence Chain

原始素材、指标、洞察、文案、排期与校验串成证据链。

05Export Vault

最终 Markdown/Word 不只是下载文件，而是归档交付物。
